# AI Video Restoration Platform

The goal of this project is to build a computer vision workflow capable of restoring blurred and degraded videos using state-of-the-art transformer-based image restoration models.

## 1. The Problem

Many videos suffer from blur, motion artifacts, compression noise, or low visual quality due to poor recording conditions, camera limitations, or transmission losses. These degradations reduce usability for content creators, media archives, surveillance systems, and downstream computer vision applications.

## 2. The Dataset

The dataset consists of video frames extracted from low-quality videos. Each frame serves as an input sample for the restoration model, while the enhanced output frames are generated by the restoration pipeline.

## 3. Features

- Video frame extraction
- Image restoration with Restormer
- Batch processing
- Video reconstruction
- Resolution benchmarking (720p vs 4K)
- Extensible architecture for multiple restoration models
